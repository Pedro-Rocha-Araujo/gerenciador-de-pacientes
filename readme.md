# Backend com mongoDb para gerenciador de pacientes 🩺 🩹 💊
### O presente respositório foi criado com o intuito de hospeda meus aprendizados como deenvolvedor fullstack
### Características do projeto:
1- O backend do projeto foi feito utilizando NodeJs e está configurado para rodar na porta 4000
2- O banco de dados utilizado na aplicação foi o MongoDb atlas
3- Os dados sensíveis do projeto foram armazenados utilizando variáveis de ambiente (dotenv)
4- A arquitetura utilizada foi a MVC (Model, view e Template)
5- O frontend do projeto foi desenvolvido em ReactJs
### Funcionalidades do sistema:
* Listagem, cadastro, edição e remoção de pacientes (CRUD)

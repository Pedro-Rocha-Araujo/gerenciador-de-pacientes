# Backend com mongoDb para gerenciador de pacientes
### O backend do projeto foi escrito em NodeJs e possúi as seguintes características:
* O backend do projeto foi feito utilizando NodeJs e Express
* O banco de dados utilizado na aplicação foi o MongoDb atlas, e aconexão foi facilitada por meio da biblioteca Mongoose
* Os dados sensíveis do projeto foram armazenados utilizando variáveis de ambiente (dotenv)
* A arquitetura utilizada foi a MVC (Model, view e Template), tendo uma pasta dedicara para a organização de Middlewares e Rotas
